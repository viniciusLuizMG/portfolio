MINHAS ESPECIALIDADES.

HTML

HTML é linguagem para estruturar conteúdo web usando que se utiliza tags como HTML, HEAD, BODY, entre outras.

CSS

CSS (Cascading Style Sheets) é usada junto ao HTML para estilizar páginas web, controlando layout, cores, fontes e outros aspectos visuais para melhorar a apresentação.

>

JAVASCRIPT

JavaScript (JS) é uma linguagem de programação utilizada para adicionar interatividade e dinamismo a páginas web, manipulando conteúdo, respondendo a eventos e criando funcionalidades avançadas.

PYTHON

Python (Py) é uma linguagem de programação versátil e de alto nível, conhecida pela simplicidade, legibilidade e vasta aplicação em desenvolvimento web, científico e automação.

PHP

PHP é uma linguagem de script amplamente utilizada para desenvolvimento web. É ideal para criar páginas dinâmicas, processar formulários e interagir com bancos de dados de forma eficiente.

XAMPP

XAMPP inclui MySQL, um sistema de gerenciamento de banco de dados amplamente usado, ideal para armazenar e manipular dados em aplicações web locais.
